# FlowIDE
Flow is a commandline based IDE/Text Editor.
## Current Features:
* Syntax Highlighting
* Clickable Menus
## Features in Progress
* Spell Checker
## Planned Features:
* Copy & Paste
* Keystroke History
* Multiple Tabs
* Saving files
* Project Files Hierarchy View
* Small code preview window
* Configurable Preferences
* Color Themes
* Syntax Themes
* Code search
* Code completion
* Refactoring
* Language support
* Intergrated Command Line
* Version control
* Debugging
