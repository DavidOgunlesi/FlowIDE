# FlowIDE
Flow is a commandline based IDE/Text Editor.
## Current Features:
* Syntax Highlighting
## Planned Features:
* Multiple Tabs
* Color Themes
* Syntax Themes
* Configurable Preferences
* Copy & Paste
* Project Files Hierarchy View
* Clickable Menus
* Small code preview window
* Saving files
* Code completion
* Refactoring
* Version control
* Debugging
* Code search
* Language support
* Intergrated Command Line
